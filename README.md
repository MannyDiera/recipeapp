# Recipe App

## Recipe application written in Vue for the JS Alpharetta meetup

### Project setup
```
npm install
```

Add edamam API credentials to .env file

```
  VUE_APP_EDAMAM_ID=""
  VUE_APP_EDAMAM_KEY=""
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
